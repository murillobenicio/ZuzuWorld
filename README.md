# 💜 Zuzu World

A romantic web project created to celebrate a special relationship through memories, photos, videos, and heartfelt messages.

The website features:

- 🎥 Video background hero section
- 📸 Interactive photo gallery
- 💌 Personalized memory cards
- 🎵 Background music
- 💜 Romantic scrapbook-inspired design
- 📱 Responsive layout for desktop and mobile devices

This project was built with pure HTML, CSS, and JavaScript as a unique digital gift and keepsake.

## Technologies

- HTML5
- CSS3
- JavaScript

## Purpose

To transform shared memories into an immersive and emotional digital experience, combining storytelling, visuals, and music in a single webpage.
